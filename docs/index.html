Methodology

Comparable enough to use.
Different enough to explain.

States set their own tests and performance labels. This project preserves those original labels, then identifies the closest available indicator of students performing at the lowest level.

01

Exact state tiers

We use the share in the state’s lowest published performance tier, such as Level 1, Novice, or Does Not Meet Expectations.

02

Other state sources

Where the state publishes only a proficiency rate or a combined non-proficient bin, we retain it with a distinct label rather than presenting it as an exact tier.

03

Federal fallback

Ed Data Express fills remaining gaps as 100 minus percent proficient. These older values provide coverage, but they are not a strict below-basic measure.

!

Do not rank without context

Assessment standards, years, and reporting structures vary. Use this dashboard to compare coverage and reported outcomes, not as a precise league table.
